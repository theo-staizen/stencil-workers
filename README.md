# Repro Repo for [Stencil#2731](https://github.com/ionic-team/stencil/issues/2731)

## Prepare stencil package

```sh
cd stencil-workers
npm i
```

## To run Webpack 4 Example

```sh
cd webpack4
npm i
npm run build
open dist/index.html
```

## To run Webpack 5 Example

```sh
cd webpack5
npm i
npm run build
open dist/index.html
```
